# Port Scanner

## About

This is my third Python cybersecurity projec that i have made and it is a port scanner.

The aim of this program is to scan a target IP address and determine whether common ports are open or closed which should show in the terminal if working.

The program uses Pythons socket library to attempt connections to a number of well known ports and then displays the results in an easy form of report.

## Features

- Scans common TCP ports
- Detects open and closed ports
- Displays common service names
- Counts total open and closed ports
- Measures scan time
- Handles invalid IP addresses
- Handles keyboard interruption

## Technologies Used

- Python
- Socket Library
